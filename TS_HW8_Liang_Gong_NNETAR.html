NNETAR (Neural Network Time Series Autoregression)

A forecasting model in R that uses a feedforward neural network to model time series data. It is implemented in the forecast package and works like an autoregressive model, using past values as inputs.

# The **square root transformation** is used to **stabilize variance** and **reduce the impact of large values** in the "Turnover" variable. This helps the **NNETAR** model learn patterns more effectively, especially if the data exhibits **heteroscedasticity** (changing variance over time). 

nn_base <- aus_retail_A3349744F %>%
  filter(Month < max(Month)-36) %>%
  model(nn = NNETAR(sqrt(Turnover)))
forecast_nn_base <- nn_base %>%
  forecast(h = 36)

print(forecast_nn_base)

Models Evaluation

Summary of the Model Performance Results

The table presents the performance comparison of three different time series forecasting models: Neural Network (NNETAR), Exponential Smoothing (ETS), and ARIMA. The metrics used for evaluation are:

  1. RMSE-based Scale-Free Error (RMSSE) Measures the root mean squared scaled error. Lower values indicate better model performance.
  2. Mean Absolute Scaled Error (MASE) Measures the accuracy relative to a naive benchmark. Lower values suggest a better model.

Key Observations:

  • Neural Network (NNETAR) performed the best, with the lowest RMSSE (1.07) and MASE (1.11), suggesting that it produced the most accurate forecasts compared to the naive benchmark.
  • ETS performed moderately well, but with higher RMSSE (1.38) and MASE (1.45) than NNETAR, indicating that it had higher forecast errors.
  • ARIMA had the worst performance, with the highest RMSSE (1.99) and MASE (2.18), showing that its forecasts were the least accurate among the three models.

Conclusion:

  • The Neural Network (NNETAR) model is the best choice for forecasting in this case, as it has the lowest errors.
  • The ETS model is a reasonable alternative but is outperformed by the neural network.
  • The ARIMA model struggles the most, likely due to the complex nature of the time series data, where nonlinear patterns may be better captured by NNETAR.

Recommendation: If model interpretability is important, ETS could be considered. However, if the priority is accuracy, the Neural Network model should be preferred for future forecasts.
